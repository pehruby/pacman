# Pac Man LED Pixel Panel Costume
Replication of awesome project originally created by [Ben Muller](https://www.hackster.io/pix3lot/pac-man-led-pixel-panel-costume-515666)

[![Pac Man costume](https://github.com/pehruby/pacman/blob/master/Photos/yt.jpg?raw=true)](https://youtu.be/wuHAoJ8ybKk "Pac Man")


* Source code is created for Atmel Studio but it can be easily used in Arduino environment.
* Controller board is build on ATMega328P chip. Arduino Uno can be used as well. Datapin 3 controls the LED string.

* 200x WS2811 LEDs (4 string of 50 LEDs connected together) were bought on eBay.
* paper parts were laser cutted from 3mm cardboard plates of size 800x500mm. DWG and PDF source files are located in Misc directory. 6 3mm cardboard 
plates of size 800x500mm are needed
